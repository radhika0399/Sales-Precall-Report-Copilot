The AI agent is created to prepare pre-call report for  Sales rep using Prospect Linkedin and Company Link


Give prompt to Agent in the format similar to below

**Input Example Prompt**


I am getting on a call with Liam Ottley from Morningside AI
Links: 
https://morningside.ai/ 
https://www.linkedin.com/in/liamottley/ 

Get me a pre-call report please to prepare for the call 



**Output Format**

It will generate the  Pre-call report using all the information of Prospect and Company and prepare the Sales rep for the questions to ask and detailed profiles to refer so as to not miss the important information and be precise in putting up questions during the call.


Why to use Agent and not ChatGPT ?

1. Agent only crawls required upto date information from given URLs and not the irreleavant data like ChatGPT
2. ChatGPT most of the times do text search and gives false information about companies as it not only uses Linkedin data but also lot of references of text on other random sites .For above example chatgpt will give you wrong data too as below.

   Put the same prompt above in ChatGPT and see the results of company summary and check below entry refering to  

  _ One UK/Scottish registered entity shows “dormant” status, which may reflect a subsidiary or registration detail. 
Endole_ refering to(https://open.endole.co.uk/insight/company/SC794539-morningside-ai-limited?utm_source=chatgpt.com)
